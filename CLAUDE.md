## Project Overview

Website for "PRO STOPY" — a podiatry cabinet in Kyiv, Ukraine, run by the owner's wife. Built with Astro, deployed on Netlify at https://prostopy.com.

**Goal:** Client acquisition from the web. Currently the only source of clients is Instagram (@podolog_datsyshyna). The site should rank in Google Search for podiatry-related queries in Kyiv and attract new clients organically.

**Content:** Bilingual — Ukrainian (default, `/`) and English (`/en/`) for expats. Service sub-pages are currently placeholder "coming soon" pages. Prices and photos are pending (will be provided by the owner).


## Architecture

**Framework:** Astro 5 with Tailwind CSS 3. Static HTML output with near-zero client JS (mobile menu toggle + review translation toggle on EN pages). TypeScript in strict mode.

**i18n:** Astro built-in i18n routing. Ukrainian is the default locale (no prefix), English at `/en/`. Translation dictionary in `src/i18n/ui.ts`, helpers in `src/i18n/utils.ts`. All user-facing text uses `t()` calls. Fallback: missing EN key → Ukrainian value.

**Routing:** File-based via `src/pages/`. UK pages at root, EN pages mirrored under `src/pages/en/`. Service sub-pages use shared `ServicePlaceholder.astro` component.

**Layout:** Single master layout at `src/layouts/Layout.astro` — sticky nav with language switcher, footer, meta/SEO tags, hreflang alternates, JSON-LD (MedicalBusiness + WebSite schemas).

**Components** (`src/components/`): `Button.astro`, `Card.astro`, `Logo.astro`, `SectionTitle.astro`, `LanguageSwitcher.astro`, `ServicePlaceholder.astro`.

**Styling:** Tailwind utilities + CSS custom properties in `src/styles/global.css` (CSS layers: base, components, utilities). Custom palette in `tailwind.config.mjs`: dark (#393E46), accent (#E5D8CC), gray (#878079), base-1 through base-6. Fonts: `prosto` (Prosto One, headings), `montserrat` (Montserrat, body).

**Static assets:** `public/` — favicons, OG image, robots.txt.

## SEO

- JSON-LD: HealthAndBeautyBusiness schema (@id, areaServed Kyiv, contactPoint, availableService for 6 services) + WebSite schema. Not MedicalBusiness — podology in Ukraine is non-medical, no medical license required.
- hreflang tags on every page (uk-UA, en-US, x-default)
- Open Graph + Twitter Card meta tags
- Sitemap auto-generated by `@astrojs/sitemap` at `/sitemap-index.xml` with i18n alternates
- `robots.txt` points to sitemap on prostopy.com
- Custom 404 page (bilingual)
- Security headers in `netlify.toml` (HSTS, CSP frame-ancestors, X-Frame-Options, etc.)
- Submitted to Google Search Console

## Quality Gate

Post-build smoke test (`scripts/smoke-test.mjs`) runs automatically on every build. Checks: all 15 pages exist, correct `lang` attributes, hreflang/JSON-LD presence, page size sanity, robots.txt integrity. Fails the build (and blocks Netlify deploy) if any check fails.

## Deployment

Netlify. Config in `netlify.toml` (build command, security headers, Content-Language headers). Node 18. The `@rollup/rollup-linux-x64-gnu` optional dependency exists for Netlify's Linux build environment.
